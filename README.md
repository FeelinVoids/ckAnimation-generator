# ckAnimation generator

[***ENGLISH README.md***](https://github.com/Felucca24/ckAnimation-generator/blob/master/README_ENG.md)

***Примеры работы внизу!***  
Эти скрипты позволяют генерировать файлы анимаций подсветки для некоторых клавиатур от Bloody. Я тестировал на Bloody B975, должно работать на похожих моделях с RGB подсветкой. Изменяя ckAnimation_algorithms.py, можно легко добавлять свои анимации, наследуя класс Algorithm.

## Использование
Можно продолжать без этого, однако автоустановка не будет работать:
Откройте скрипт ckAnimation_generator.py блокнотом и убедитесь, что пути к папке скриптов и файлу Setting.ini KeyDominator'а правильны. Можно попробовать скопировать их и вставить в окне win+r и нажать Enter. Если папка и файл открылись, то всё хорошо. Если нет - найдите, куда установлен KeyDominator, а именно папка NumberPadAtRight и файл NumberPadAtRight\Setting.ini и исправьте пути в скрипте. Разделитель - \\\\. После этого можно продолжать.  

Запустите скрипт с помощью Python 3.
Например, через консоль.  

    Shift+Правая кнопка мыши, находясь в папке -> Открыть окно команд
    
    python ckAnimation_generator.py

Будет предложено ввести имя файла, но можно это пропустить, нажав Enter.  
После этого следует выбрать, какой из алгоритмов анимаций вы хотите использовать, для этого введите цифру - номер алгоритма в отображенном списке и нажмите Enter.  
После того, как анимация сгенерировалась, скрипт предложит установить её. Введите цифру от 0 до 9 для установки анимации на сочетание с Fn. Созданный файл будет скопирован в папку анимаций Bloody и настройки будут изменены. Это можно пропустить, тогда файл просто останется на введёном пути, или в папке со скриптом.
